**Card** — primary content container with soft rounding and a cool shadow.

```jsx
<Card accent="signet" elevation="md">
  <h3>Tag der offenen Tür</h3>
  <p>Lernen Sie die HAK kennen.</p>
</Card>
<Card interactive accent="petrol">…</Card>
```

`accent`: `petrol`/`sky`/`red`/`amber`/`signet` (four-colour bar). `elevation`: none→lg. `padding`: none→lg. Set `interactive` for hover lift.
